TowerFall Version: 8-Player (tested on 1.3.2.0)
Platform: Windows

## How to patch the game

1. (Optional) Backup your TF-8P so you don't have to redownload it to restore the original version
2. Open a command prompt and navigate to where you've unzipped the patcher
3. Run Wizard.exe with the path to the TF-8P you wish to patch. For example:

./Wizard.exe ~/Games/TF-8P
